What We Do

We help students turn programming interest into real work: learning the fundamentals, building useful school projects, and testing skills through contests, game jams, and collaborative challenges.

Learn

Structured workshops, peer feedback, and lessons in Python, C++, Swift, responsible AI use, and the habits behind steady technical progress.

Build

Contribute code, interface design, testing, and planning on projects like the RHS Companion App, HSR Battlegrounds, and tools for other clubs. You will also be learning to use industry standard software and workflows like Git+Github, Xcode, Visual Studio Code, Figma, Blender and more.

Compete

Practice for CCC-style programming contests, game jams, hackathons, and esports collaborations that sharpen problem-solving under real constraints.
